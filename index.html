Tecnologias

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação que define a estrutura e o significado do conteúdo de uma página da web. É o componente básico da web e a base para a construção de sites

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo que define a aparência de páginas web. É uma das principais linguagens da web e é usada em conjunto com HTML

JS

JavaScript (JS) é uma linguagem de programação usada para criar páginas web interativas. É uma das linguagens mais populares do mundo

GIT HUB

GitHub é uma plataforma online que permite armazenar, compartilhar e colaborar na criação de códigos. É considerada uma "rede social de código aberto" ou "rede social dos programadores

Photoshop

Photoshop é um software de edição de imagens, design gráfico e criação de arte digital. Ele é desenvolvido pela Adobe e é considerado um dos editores de imagem mais famosos.

Bootstrap

Bootstrap é um framework front-end que permite criar sites e aplicações web responsivas. Ele é gratuito, de código aberto e usa HTML, CSS e JavaScript.
